## Running Documentation Locally

## Quick Start

```bash
cd docs
uv run mkdocs serve
```

Then open: **http://localhost:8000**

## Setup (First Time)

If you need to set up the environment, first [install uv](https://docs.astral.sh/uv/getting-started/installation/):

```bash
cd docs
uv sync
uv run mkdocs serve
```

The development server includes live reloading — any changes to markdown files will automatically refresh the browser.
